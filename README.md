# notebooklm-sql-data-analysis
Este repositório contém um material focado na aplicação da linguagem SQL em contextos de Análise de Dados. O projeto utiliza o NotebookLM como ferramenta central para organizar conhecimentos que abrangem desde os fundamentos básicos até técnicas avançadas de consulta e modelagem de dados.

---

## 🎯 Contexto e Objetivos

### 📌 Contexto

O SQL (Structured Query Language) é a linguagem universal de consulta para bancos de dados relacionais e uma ferramenta indispensável para profissionais da área de dados.

Este projeto foi criado para estruturar o aprendizado de SQL com foco em análise de dados, explorando como extrair insights relevantes e resolver problemas de negócio a partir de dados estruturados.

---

### 🎯 Objetivos

Este material foi desenvolvido com os seguintes objetivos:

* **Dominar os Fundamentos:** Compreender as principais categorias de comandos SQL: DDL, DML, DCL e TCL
* **Preparação para o Mercado:** Resolver problemas e questões comuns em entrevistas para Data Analyst
* **Aplicação de Técnicas Avançadas:** Utilizar CTEs, subqueries, Window Functions e JOINs em análises complexas
* **Gerenciamento de Banco de Dados:** Entender conceitos práticos de PostgreSQL, normalização e integridade de dados
